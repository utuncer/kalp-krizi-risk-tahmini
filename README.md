# Projedeki Veriler Ne Anlama Geliyor?

| Sütun | Türkçe Karşılığı | Açıklama |
| :--- | :--- | :--- |
| Age | Yaş | Hastanın yaşı |
| Sex | Cinsiyet | M = Erkek, F = Kadın |
| ChestPainType | Göğüs Ağrısı Türü | ATA = Atipik Angina, NAP = Non-Anjinal Ağrı, ASY = Asemptomatik, TA = Tipik Angina |
| RestingBP | Dinlenme Kan Basıncı | Hastanın dinlenme halindeki sistolik kan basıncı (mmHg) |
| Cholesterol | Kolesterol | Serum kolesterol değeri (mg/dL) |
| FastingBS | Açlık Kan Şekeri | Açlık kan şekeri|
| RestingECG | Dinlenme EKG Sonucu | Normal, ST (ST-T dalgası anormalliği), LVH (sol ventrikül hipertrofisi) |
| MaxHR | Maksimum Kalp Hızı | Egzersiz sırasında ulaşılan en yüksek kalp atış hızı (bpm) |
| ExerciseAngina | Egzersiz Anjinası | Egzersizle tetiklenen göğüs ağrısı |
| Oldpeak | ST Depresyonu | Egzersizin dinlenmeye kıyasla neden olduğu ST segment çökmesi (mV) |
| ST_Slope | ST Eğimi | Egzersiz ST segmentinin eğimi |
| HeartDisease | Kalp Hastalığı | 1 = Kalp hastalığı var, 0 = Yok |
